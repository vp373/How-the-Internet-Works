The Web

W3C

W3C stands for the World Wide Web Consortium. This is an international group which develops the standards for the web. It is led by the inventor of the web Tim Berners-Lee, and their goal is to help fully capitalize on the benefits of the web for all.

W3C is a crucial part to the web infrastructure. They are responsible for setting forth the rules and protocols that will ensure the growth of the web and make it more accessible for all!

An example of some of the work that W3C does includes the Open Standards Principles. This is a model that empowers individuals to develop new technologies and innovate further.

Learn more about W3C

HTML

HTML stands for Hypertext Markup Language. This is the standard markup language for web pages and can be paired with CSS and javascript for more visually enhanced web pages.

HTML is a powerful language which enables us to build out webpages. It was developed by the founder of the web and has many unique features which enable us to build out complicated web pages.

CSS

CSS stands for cascading style sheets. It is a essentially just a way to spice up your webpages and add style to it. It is designed to improve accessibility, presentation characteristics, and reduce redundancy. It was developed by W3C nearly 25 years ago!

CSS allows you to fully customize the presentation of your webpage- from colors, to spacing, to fonts, and much more! It is often misunderstood that CSS is paired to be used only with HTML, but in fact it can be used with any XML based markup language!
