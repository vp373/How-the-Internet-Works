IP Addresses

An IP Address is a numerical identifier that is assigned to all devices that are connected to a network. IP stands for internet protocol- you can think of the IP address as the address of your device. IPv4 is a 32 bit number, however, due to the rapid expansion of the internet there are limited addresses left. For this reason, we have been transitioning to IPv6 which is a 128 bit number and allows for many more addresses.

IP Addresses are crucial to how the internet functions. When servers look for addresses to send data packets to, the IP address is what distinguishes your device and ensures delivery to the right place.

Routing

Domains are computers/devices that share difference types of data. These domains can be administered and accessed with a common set of rules.

There are often many confusing terms that sound similar that are also used interchangeably including Domains, Domain Names, URLs and Web addresses, but what are all of these? Simply put, a domain name is your web address, and your web address is a URL. We use these domains as a type of address to identify where your webpage lives.

An example of a domain would be www.google.com or www.github.com. These URL's point to a certain web address which is stored in a server. This is more practical than remembering the long numerical IP address for your website.
