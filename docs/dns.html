Domain Name System

DNS

While internet browsers interact through IP addresses, humans cannot efficiently use those numerical labels. Instead we use domains. Domains can include sites like google.com or yahoo.com

The Domain Name System is a crucial part of the internet which allows humans to more effectively utilize the internet.

An example of how DNS works is a comparable to a phonebook or directory. For instance, when you type in google.com, DNS servers will look up the IP address for google.com and redirect you there automatically! The image to the right shows the concept of DNS at a high level.
