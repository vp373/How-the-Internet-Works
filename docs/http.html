Hypertext Transfer Protocol

HTTP & HTTPS

HTTP is a protocol that is used to transfer data over a network. Essentially, this protocol is a set of rules that dictates how information is transmitted on the World Wide Web. There are 2 main types of HTTP messages; requests and responses. HTTPS is the same thing as HTTP, but with encryption. The extra "s" at the end stands for secure. HTTPS is much more secure because it uses SSL to encrypt the data that is being transferred over that network.

HTTP & HTTPS is what allows most data to be transferred over the internet. Often times we may take this for granted, but it is an important topic to understand. This is the set of rules standard rules which enables different sites to communicate with each other over a network.
